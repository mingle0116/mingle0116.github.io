Windows11 如何搭建个人的博客及部署到GitHub

Tips

下载安装 node.js (官网安装) https://nodejs.org
然后cmd +node
npm ：包管理器 node package management

“node -v”：查看node 的版本
“npm root -g”查看全局包的路径地址
“npm -v”：查看npm的版本

“npm install -g cnpm –registry-https://registry.npm.taobao.org”:安装cnpm的
“cnpm install -g hexo-cli”:国内镜像的cnpm 下载hexo框架

基本的配置：Windows 11

基本流程

  1. 安装node.js
    通过npm 安装，先安装cnpm淘宝的镜像，再安装hexo
    “npm install -g cnpm –registry-https://registry.npm.taobao.org”:安装cnpm的

  2. 安装hexo
    “cnpm install -g hexo-cli”:国内镜像的cnpm 下载hexo框架

  3. 通过hexo搭建个人博客
    如果在cmd中进入磁盘的命令不行的，可以试一试这个语句：
    “cd /d D:”返回：“cd ../”.
    “md blog”:建立名为blog的文件夹
    “hexo init”:hexo 的初始化
    “dir”：查看当前文件夹的文件
    “hexo s”：启动博客

  4. 编写博客 用到的工具：vim 语言: markdown
    “http://localhost:4000/”：本地端口4000 开始的
    “hexo n “tittle of article””:写新的博文
    “good bye”:退出
    “givm tittle.md”:使用gvim 打开md文件
    按住esc ，然后输入“：wq”
    “hexo clean “:清除一下
    “hexo g”:重新生成
    “hexo s ”：启动博客

  5. 把博客部署到github上面去
    在github上面建立一个新创库，然后命名为github的用户名.github.io
    “cnpm install hexo-deployer-git –save”:安装插件
    然后再配置一下 _config.yml 文件中，找到deloyment
    添加如下内容：
    type:git
    branch:master
    repo:(建立新创库的地址)https://github.com/github'username/mingle0116.github.io.git

“hexo d”：hexo 部署到远端
在此之前要 要在gitbash配置 初始化
语句是“git config user.name “github’username””
“git config user.email “xxx@””

  1. 更换主题
    首先找到主题模板，然后克隆到theme 文件下就可部署
    “git clone address.git themes/theme’sname”
    然后再_config.yml文件中的theme:landspace 改为 theme’sname即可

部署流程

部署前预览的流程：
hexo clean
hexo g
hexo s

部署远端的流程：
hexo d

node.js 的环境配置的 相关内容 http://t.csdn.cn/MfY9
